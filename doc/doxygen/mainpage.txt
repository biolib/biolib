/*! \mainpage BioLib API Documentation
 *
 *  These pages describe the BioLib API as generated from
 *  the source code. This API can be used directly from Perl and 
 *  Ruby.
 *
 *  Using the mapped functions is straightforward. For example
 *  the method read_reading is listed in the Staden io_lib API as:
 *
 *  \code
 *    // C method:
 *    Read *read_reading(char *fn, int format);
 *  \endcode
 *
 *  In Perl the method can be accessed with:
 *
 *  \code
 *    // Perl mapping:
 *    $result = staden_io_lib::read_reading($procsrffn,$staden_io_lib::TT_ANY);
 *  \endcode
 *
 *  In Ruby the method can be accessed with:
 *
 *   \code
 *     // Ruby mapping:
 *     result = Biolib::Staden_io_lib.read_reading(procsrffn,Biolib::Staden_io_lib::TT_ANY)
 *   \endcode
 *
 *  Thanks to BioLib's use of SWIG. The documentation of the C API is
 *  generated straight from the source code, so it is most up-to-date. 
 *
 *  For more information on the BioLib project and language mappings, see http://biolib.open-bio.org/
 */
